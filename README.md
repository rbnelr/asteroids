# asteroids
 asteroids with c++ and opengl
 
 
 build with build.bat
  
  'build.bat vs' for msvc compiler (cl.exe) (needs to be in path env var)
  
  'build.bat gcc' for gcc compiler (needs gcc bin path in env var called 'GCC')
